# Object-Oriented Taiwan High Speed Rail Booking System
	Computer Programming II — Homework Assignment #13
	Function & Requirements:
	1.	Booking: Add a new record into the binary file Reservation details.dat (randomly generate a reservation number).
	2.	Booking History: Enter the contact person's ID number and reservation number to retrieve an existing reservation.
	  Cancellation: Cancel the reservation
	  Reduce: Decrease the number of passengers
	3.	Each function (include the main function) contains at most 80 instructions.
	4.	Global variables, except adultTicketPrice, concessionTicketPrice and departureTimes, are not allowed.
